# 📊 E-Commerce Sales & Customer Analytics Dashboard

📊 End-to-end business intelligence project analyzing e-commerce sales, customer behavior, and product performance using Power BI.

---

## 🚀 Business Problem
E-commerce companies often struggle to understand customer behavior, identify top-performing products, and track revenue trends effectively.

## 💡 Solution
This project presents an interactive Power BI dashboard that analyzes sales, customers, and product performance to support data-driven decision-making.

## 📊 Impact
- Improved visibility into revenue trends  
- Identified high-value customers  
- Highlighted top-performing product categories  

---

## 📁 Dataset Description

The project uses three primary datasets:

### 🧑 Customers Dataset
- Customer ID  
- Customer Name  
- Location  
- Demographics  

### 📦 Orders Dataset
- Order ID  
- Customer ID  
- Order Date  
- Quantity  
- Sales Amount  

### 🛍️ Products Dataset
- Product ID  
- Product Name  
- Category  
- Price
