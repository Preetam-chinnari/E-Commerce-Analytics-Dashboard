# 📊 E-Commerce Sales & Customer Analytics Dashboard

## 🔍 Overview
This project focuses on analyzing e-commerce sales data to uncover insights related to customer behavior, product performance, and revenue trends. The dashboard is built using **Power BI** and integrates multiple datasets to provide actionable business insights.

---

## 📁 Dataset Description

The project uses three primary datasets:

### 🧑 Customers Dataset
- Customer ID
- Customer Name
- Location
- Demographics

### 📦 Orders Dataset
- Order ID
- Customer ID
- Order Date
- Quantity
- Sales Amount

### 🛍️ Products Dataset
- Product ID
- Product Name
- Category
- Price

---

## 🛠️ Tools & Technologies
- **Power BI** – Data visualization & dashboard creation  
- **SQL** – Data querying and transformation  
- **Microsoft Excel** – Data preprocessing  
- **Python (optional)** – Data cleaning and analysis  

---

## 📊 Key Features

- 📈 Revenue Analysis Dashboard  
- 👥 Customer Segmentation Insights  
- 📦 Product Performance Tracking  
- 🔁 Repeat Customer Analysis  
- 📉 Order Trends & Seasonal Insights  
- 📊 KPI Metrics:
  - Total Revenue  
  - Average Order Value (AOV)  
  - Customer Retention Rate  
  - Cancellation Rate  

---

## 📌 Project Workflow

1. Data Collection from multiple sources  
2. Data Cleaning & Preprocessing (Excel/SQL)  
3. Data Modeling in Power BI  
4. Dashboard Design & Visualization  
5. Insight Generation
